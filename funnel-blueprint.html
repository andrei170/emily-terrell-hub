Tech Stack (recommended, swappable)

  • CMS / Course host: Kajabi (preferred, all-in-one) or Teachable
  • Landing pages: Custom HTML (this package) or Webflow for the course page
  • Email: Kit / ConvertKit (clean for solo creators) or Kajabi-native
  • Quiz / Scorecard: ScoreApp (preferred for scoring + segmentation) or Typeform
  • Payment: Stripe (course checkout)
  • Video host: Wistia (course intro) + YouTube (long-form content)
  • Analytics: GA4 + Meta Pixel + Microsoft Clarity (heatmaps)
  • Scheduling: Calendly (1:1 coaching bookings, already in use at coachemilyterrell.com)
  • Speaker assets: Canva Pro (QR slide templates, speaker deck updates)
  • Community (future): Skool or Circle if cohort model added downstream

Integration Points

  1. Scorecard → Email: ScoreApp triggers Kit on quiz completion with score tag
  2. Email → Course: Kit triggers Kajabi enrolment on purchase webhook
  3. Course → 1:1 Coaching: Course graduates routed to 1:1 coaching application on completion of module 5
  4. Speaker QR → Scorecard → Post-event sequence: QR routes to scorecard; score triggers the correct post-event nurture track
  5. All events → GA4 + Meta Pixel via GTM: full funnel attribution across blog, IG, and course
  6. Blog in-line CTAs → Scorecard: updated blog posts include scorecard link at conclusion of each post

Owner Map

  • Emily: filming (VSL intro + speaker content), course content delivery, live Q&A if added, top-line decisions on copy + brand, speaking engagements
  • AG1 (Andrei): everything else - landing page build, email sequence writing, automation setup, scorecard build, speaker QR template, content production cycles, ads management, weekly reporting

Failure Modes + Mitigations

  • Course content filming delays → use existing conference talk excerpt or IG reel as Phase 1 placeholder; replace with custom VSL by end of Phase 1
  • Speaker QR adoption → start with a single upcoming talk; test before rolling out to all events
  • Scorecard over-segmentation → start with 2 tiers (course-ready vs 1:1-ready); expand only after 500+ quiz completions
  • Blog CTA rollout → prioritise the 10 highest-traffic blog posts first; update the rest in batches

KPIs (90-day)

  • Phase 1: AI Realtor Mentorship page live, 30+ buyers in first 30 days
  • Phase 2: 500+ scorecard completions, 8-12% IG-to-email conversion from bio-link update
  • Phase 3: 40+ short-form posts shipped, 1 speaker series episode published
  • Phase 4: Meta CPL under $35, ROAS 3x by week 12
  • Year-1 floor target: $150-380K incremental on top of existing 1:1 coaching and speaking revenue
